# 💼 Personal Portfolio Website

This is a responsive personal portfolio website built using HTML, CSS, and Bootstrap. It showcases my skills, projects, and contact information in a clean and modern design.

The website is designed to represent my work as a Python Developer and highlight my experience across multiple domains like web development, data science, automation, and more.

Link : https://karthikeyank-25.github.io/Karthikeyank-25-Portfolio_Website-2026/

## 🚀 Features

* 🏠 Home section with introduction
* 👤 About section describing my background
* 🛠️ Skills section with categorized expertise

## ⚙️ How It Works

* The website is divided into sections:

  * **Home** → Introduction
  * **About** → Personal details
  * **Skills** → Technical skills and tools
  * **Projects** → Work showcase
  * **Contact** → Communication details

* Uses Bootstrap grid system for responsive layout

* Smooth scrolling navigation between sections

* Popup image gallery for project previews
